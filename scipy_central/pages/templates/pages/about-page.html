About

SciPy Central is a collection of code snippets, modules and links for solving scientific problems with SciPy and related Python tools.

Back in October 2010 and again in April 2011 there were discussions on the SciPy mailing list about creating a File Central website. This site came about from all the interesting ideas discussed in those threads.

You can download the code that runs this site from GitHub. The code is BSD licensed and relies on a few other projects, which are mostly BSD licensed as well. Eventually we will have all the code and link contributions that have been made to this site available for download in a convenient ZIP file.

If you'd like to suggest improvements to the site (anything!) please create a new issue on the GitHub site. We'll do our best to fix any bugs right away, or to consider adding new features. If you prefer to contact us by email, please write us at admin@scipy-central.org.
